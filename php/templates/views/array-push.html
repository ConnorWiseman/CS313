Array Push

Ways to push an element onto the end of an Array.

Preferred

array_push($array, 1, 2, 3);

The array_push function can push multiple values onto a given Array and is therefore the most concise and versatile option.

Acceptable

$array[] = 1;
$array[] = 2;
$array[] = 3;

Using implicit Array assignment syntax is an acceptable alternative if and only if the Array already exists.

Avoid

$array = $array + [ 1, 2, 3 ];

Arrays can be concatenated to one another with the + operator, but this is unsuitable for pushing single values onto an Array and is not as explicit as array_push.

Anti-pattern

$array[0] = 1;
$array[1] = 2;
$array[2] = 3;

Directly assigning to Array indices introduces unwanted side effects, like arbitrarily changing the Array length to accomodate the final index. Do not do this.

Anti-pattern

$array = [ 1, 2, 3 ];

array_map(function($element) {
  // $element
  // not possible to access index through `array_map()`
}, $array);

Using array_map to iterate over an Array is a misuse of the function. Furthermore, it is impossible to access the Array index through array_map. There are better alternatives.
