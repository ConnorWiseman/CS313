Associative Array Remove Value

Ways to remove key-value pairs from an associative Array.

Preferred

$array = [
  'key' => 'value'
];

unset($array['key']);

The unset() function will remove a key from an associative Array.

Avoid

$array = [
  'key' => 'value'
];

$array = array_diff($array, [
  'value'
]);

array_diff() is a function that returns all the elements in the first Array whose value does not exist in any of the other Arrays. array_diff() can remove multiple key-value pairs from an associative Array, so do not use it unless your use case specifically requires it.
