Associative Array Initialization

Ways to initialize an associative Array.

Preferred

$array = [
  'key' => 'value'
];

The most concise way of initializing an associative Array. Explicit and safe to use as of PHP 5.4.

Acceptable

$array = array(
  'key' => 'value'
);

array() is a language construct in PHP, not a constructor, so it isn't called with the new key word. PHP's language constructs are case-insensitive. In this case, they're also more verbose than the associative Array literal syntax with square brackets.

Anti-pattern

$array[/* first key here */] = /* first value here */;

Using direct bracket assignment syntax to initialize an associative Array with its first value is not a safe operation. If the given associative Array already exists, this will simply create a new key-value pair on the existing associative Array or, worse, add an associative Array key to an ordinary zero-indexed Array. More often than not, that behavior is not desired.
