Associative Array Contains

Ways to determine if an associative Array contains a key-value pair.

Preferred

array_key_exists('one', $array);

The array_key_exists function returns true if a given key exists in an associative Array.

Acceptable

key_exists('one', $array);

The key_exists function is an alias for array_key_exists. For consistency's sake, favor the former.

Avoid

array_search('one', array_keys($array)) !== false;

array_search will return the key of the given value in an associative Array if found, and false if not found. Performing an extra comparison is additional, unnecessary cognitive overhead. Just use array_key_exists.
