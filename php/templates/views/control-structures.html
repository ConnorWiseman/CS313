Control Structures

PHP's control structures.

if statement

if (/* condition */) {
  // logic if condition is true
}

if/else statement

if (/* condition */) {
  // logic if condition is true
} else {
  // logic if condition is false
}

if/else if/else statement

if (/* condition one */) {
  // logic if condition one is true
} else if (/* condition two */) {
  // logic if condition one is false but condition two is true
} else {
  // logic if neither condition one nor condition two are true
}

switch statement

switch(/* value */) {
  case 'one':
    // logic to perform if value is 'one'
    break; // break from switch after logic is through
  case 'two':
  case 'three':
    // switch supports fallthrough:
    // logic to perform if value is either 'two' or 'three'
    break;
  default:
    // logic to perform is no case statement matches the value
}

for loop

for ($i = 0; $i < 5; $i++) {
  // initialize iteratior variable $i to zero
  // increment $i after each loop
  // execute loop repeatedly until the condition $i < 5 evaluates to false
}

foreach loop

foreach($array as $element) {
  // logic to perform with every element in an Array
}

while loop

while (/* condition */) {
  // logic to execute repeatedly while condition is true
}

do... while loop

do {
  // logic to execute repeatedly while condition is true
} while (/* condition */);

The primary difference between the while loop and the do... while loop is that do... while will always execute at least once, since the condition is not evaluated until after loop execution.

Do not use

PHP, being PHP, has introduced awkward and completely unnecessary alternative syntax for its control structures. Never use it.
