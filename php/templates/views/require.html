Require

Ways to modularize PHP code.

include pulls the contents of the target PHP script into current scope and evaluates it. Note that classes and functions are always placed into global scope.

If a given script file cannot be found, include will continue script execution with a warning, but require will cause a fatal error. If your program cannot continue in the absence of a particular file, prefer require over include.

require

require 'myFile.php';

require_once

require_once 'myFile.php';

require_once will only require the target file if it has not already been required in the current scope. This behavior is useful for scripts that define functions.
