Include

Ways to modularize PHP code.

include pulls the contents of the target PHP script into current scope and evaluates it. Note that classes and functions are always placed into global scope.

By The Way

There are few reasons to prefer include over require.

include

include 'myFile.php';

include_once

include_once 'myFile.php';

include_once will only include the target file if it has not already been included in the current scope. This behavior is useful for scripts that define functions.
