Function Declaration

Ways to define your own functions.

By The Way

In PHP, function and class definitions are almost always placed in global scope. This can cause issues when including the same file more than once.

Anonymous Function

function() {
  // an anonymous function
};

Anonymous functions are functions that have no name. They're usually passed to other functions as arguments or assigned to variables to enable use, because anonymous functions cannot be executed on their own (outside the immediately-invoked function expression pattern).

Named Function

function returnsTrue() {
  return true;
};

PHP7 Type Hinting

function returnsTrue() : bool {
  return true;
};
function repeatAfterMe(string $str) : string {
  return $str;
};

PHP7 permits type hinting of function parameters and return types. If you are working in PHP7, you may as well take advantage of this feature. All examples of PHP functions will use type hinting.

Rest/"splat" operator

function addInts(int ...$n) : int {
  return array_reduce($n, function(int $carry, int $item) : int {
    return $carry + $item;
  }, 0);
};

The rest operator ... (called the "splat" operator in languages like Ruby) is used to describe variadic arguments in function definitions. The function above takes none, one, or many arguments and makes use of array_reduce() (PHP Docs) to return the sum of all arguments.

addInts(); // 0
addInts(1); // 1
addInts(1, 2); // 3
addInts(1, 2, 3); //6
