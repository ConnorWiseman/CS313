Array Contains

Ways to determine if an Array contains a value.

Preferred

in_array(1, $array, true);

The in_array function returns true if a given value exists in an Array. The final parameter determines which type comparison to use; always favor true for strict type comparison.

Avoid

array_search(1, $array, true) !== false;

array_search will return the index of the given value in an Array if found, and false if not found. Performing an extra comparison is additional, unnecessary cognitive overhead. Just use in_array.
