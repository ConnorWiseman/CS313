Array Initialization

Ways to initialize an empty Array.

Preferred

$array = [];

The most concise way of initializing an empty Array. Explicit and safe to use as of PHP 5.4.

Acceptable

$array = array();

array() is a language construct in PHP, not a constructor, so it isn't called with the new key word. PHP's language constructs are case-insensitive. In this case, they're also more verbose than the Array literal syntax with square brackets.

Avoid

$array = (array) null;

Coercing null into an Array will yield an empty Array, but the syntax is awkward and not easily recognized. At the same time, there is no benefit to this approach. Avoid using.

Anti-pattern

$array[] = /* first value here */;

Using direct bracket assignment syntax to initialize an Array with its first value is not a safe operation. If the given Array already exists, this will simply push a new element onto the end of the existing Array. More often than not, that behavior is not desired.
