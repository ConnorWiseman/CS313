Array Length

Ways to count the number of elements in an Array.

Preferred

count($array);

The count function returns the number of elements in an Array. Simple, straightforward.

Avoid

sizeof($array);

sizeof is a PHP alias for count, but carries unfortunate implications for developers with a background in C or C-like languages. Just use count.

Anti-pattern

new ArrayObject($array)->count();

ArrayObject is a useful wrapper class for iterating over objects that are not already like Arrays. Using it on an Array is pointless.
